DeepL API 번역기 설명서

무료 DeepL API를 사용하여 텍스트를 번역합니다.

다음 기능들을 제공합니다:

  • 번역: 텍스트를 번역합니다. (엔터키)
  • 복사: 번역된 텍스트를 복사합니다.
  • 초기화: 입력 및 출력 필드를 초기화합니다.
  • 언어변환: 입력 및 출력 언어를 전환합니다. 영어-한국어
DeepL API 무료 요금제-> 한 달에 50만자 까지 번역 가능합니다.
